# code-challenge
Project Structure:
 
This project is configured with maven and using JDK 1.8.

 
To Build Project:
 
	Execute the following command inside project root folder:
 
    	mvn clean install

 
To Run Project:

	Execute the following command inside project root folder:

		mvn spring-boot:run
    
This project runs on default port - 8989 and the context path - /challenge
